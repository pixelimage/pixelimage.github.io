piSocials -- ソーシャルボタン埋め込みjQueryプラグイン

FacebookやTwitterなどのソーシャル関連ボタンを、まとめて簡単に埋め込むことができる、jQueryプラグインです。

使い方

  1. ライブラリのダウンロード
  2. 「設定」でカスタマイズを行う
  3. 「コードの取得」でコードを取得
  4. 自身のサイトへ組み込み

ダウンロード

ダウンロード ( ZIP ) or ソースコード ( GitHub )

設定

シェアするURLを入力します。
何も入力しない場合は、ページごとのURL ( window.location.href ) が使用されます。例えば、ブログのURLをシェアしたいときはブログのURLを入力し、記事ごとにシェアしたいときは、空欄にしてください。

シェアするタイトルを入力します。
何も入力しない場合は、ページごとのタイトルタグの値 ( document.title ) が使用されます。例えば、ブログタイトルをシェアしたいときはブログのタイトルを入力し、記事ごとにタイトルをシェアしたいときは、空欄にしてください。
